Class Logger

An object representing a simple logger that outputs to a Function's console. Will enable or disable info and warn logging based on the presence or absence of a TWILIO_DEBUG environment variable. Also tags each log message with a label representing the component, system, or app that is logging the message.

Usage: const logger = new Logger(context, 'my-app');

param context

A Serverless Context with debug flag environment variables.

param label

A string describing the component, system, or app that logging is being performed for.
